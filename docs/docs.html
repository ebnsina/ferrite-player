Documentation

One <ferrite-player> tag for HLS, DASH, MP4, and live. Shaka powers adaptive streaming and loads only when a manifest needs MSE.

Install

pnpm add ferrite-player

Then import it once — importing registers the <ferrite-player> custom element:

import 'ferrite-player';

Or straight from a CDN, no build step:

<script type="module" src="https://esm.sh/ferrite-player"></script>

Quickstart

<ferrite-player src="https://cdn.example.com/master.m3u8"></ferrite-player>

The default skin — play, scrubbable seek, volume, quality, captions, PiP, fullscreen, live badge, keyboard shortcuts — is included. Point src at any .m3u8, .mpd, or .mp4; the player picks native playback where it can and lazy-loads Shaka for the rest.

Attributes

AttributeTypeDescription
srcstringThe stream/file URL. Changing it reloads.
posterstringImage shown before playback.
autoplaybooleanStart playback on load (auto-muted, per browser policy).
mutedbooleanStart muted.
crossoriginstringSets the underlying <video> crossorigin (for captions/DRM).

JavaScript API

const player = document.querySelector('ferrite-player');

// Load with options — live, low-latency, DRM:
await player.load('https://cdn.example.com/manifest.mpd', {
  lowLatency: true,
  drm: { servers: { 'com.widevine.alpha': 'https://license…' } },
});

player.play();          // returns a Promise
player.pause();
player.src;             // current source (getter)
player.src = url;       // setter → loads
MethodDescription
load(src, opts?)Load a source with optional LoadOptions (live, lowLatency, drm, autoplay).
play() / pause()Control playback.
on(cb)Subscribe to state changes; returns an unsubscribe function.
stateA read-only snapshot of the current PlayerState.

State & events

Subscribe with on() and read player.state — a plain snapshot updated on every change:

const off = player.on(() => {
  const s = player.state;
  console.log(s.currentTime, s.duration, s.paused, s.live, s.qualities);
});
// later: off();

PlayerState includes: src, engine ('native'|'mse'), paused, waiting, currentTime, duration, buffered, volume, muted, rate, fullscreen, pip, live, atLiveEdge, qualities, currentQuality, textTracks, currentText, and error.

Theming

Style with CSS custom properties — no shadow-DOM piercing needed:

ferrite-player {
  --fp-accent: #ff3e00;   /* controls + progress */
  --fp-radius: 14px;      /* corner radius */
}

Frameworks

It's a custom element, so it works everywhere. Import once at your app entry, then use the tag.

// React
import 'ferrite-player';
<ferrite-player src={url} />

// Vue / Svelte — same: import once, use <ferrite-player src=…>

In React, add ferrite-player to your JSX intrinsic elements types (or cast) for full typing.

Live streaming

Pass live/lowLatency at load. A live badge appears; when the viewer drifts back, it dims — one click jumps to the live edge.

await player.load('https://cdn.example.com/live.m3u8', { lowLatency: true });

DRM

Pass license servers (and a FairPlay cert for Safari) at load time — nothing is hardcoded in the bundle.

await player.load(url, {
  drm: {
    servers: {
      'com.widevine.alpha': 'https://license.example.com/widevine',
      'com.microsoft.playready': 'https://license.example.com/playready',
    },
    fairplay: { licenseUrl: 'https://…/fps', certificateUri: 'https://…/cert' },
  },
});

Browser support

Modern evergreen browsers. HLS plays natively on Safari/iOS; DASH and non-native HLS use Shaka (MSE + EME). MP4 plays natively everywhere. FairPlay on Safari, Widevine/PlayReady elsewhere.
