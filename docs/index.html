Why Ferrite Player

Everything a modern player needs, nothing it doesn't.

Web standards only — a custom element you can drop into any page or framework.

One API, any format

Point src at .mp4, .m3u8, or .mpd. It plays natively where the browser can — Safari/iOS HLS, MP4 — and lazy-loads the adaptive engine for DASH, non-native HLS, and low-latency live.

Tiny core

The adaptive engine ships as a separate chunk and only downloads when a manifest requires Media Source Extensions. MP4 and native HLS pay nothing for it.

HTML-first

A real custom element. Works in React, Vue, Svelte, or plain HTML — no wrapper.

Live & low-latency

A live badge with one-click jump-to-edge, plus a low-latency mode.

DRM built in

Widevine, PlayReady, and FairPlay — pass license servers at load time, never hardcoded.

Quickstart

Add the script, use the tag.

  1. 1

    Import once

    Registers the <ferrite-player> custom element globally.

  2. 2

    Drop in the tag

    Point src at any stream. The default skin comes included.

  3. 3

    Theme & script it

    CSS custom properties for looks; a small JS API for control.

<!-- 1 · import (or: pnpm add ferrite-player) -->
<script type="module"
  src="https://esm.sh/ferrite-player"></script>

<!-- 2 · use the tag -->
<ferrite-player
  src="https://cdn.example.com/master.m3u8"
></ferrite-player>
