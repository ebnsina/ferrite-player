Part of the Ferrite video stack

One tag for any video.

A framework-agnostic web-component player for HLS, DASH, MP4, and live. Shaka powers adaptive streaming — but only downloads when a manifest actually needs it, so the core stays tiny.

Why ferrite-player

Everything a modern player needs, nothing it doesn't.

▶

One API, any format

Point src at .mp4, .m3u8, or .mpd. It plays natively where it can and lazy-loads Shaka for the rest.

⚡

Tiny core

The core bundle is a few kB gzipped. Shaka ships as a separate chunk that loads only when a stream needs MSE.

◆

HTML-first

A real custom element. Drops into React, Vue, Svelte, or plain HTML — no framework, no wrapper.

●

Live & low-latency

DVR live streams with a live badge and one-click jump-to-edge, plus low-latency mode via Shaka.

🔒

DRM built in

Widevine, PlayReady, and FairPlay — pass your license servers at load time; nothing hardcoded.

🎨

Themeable & accessible

Style with CSS custom properties, keyboard-operable with ARIA, and reduced-motion aware.

Quickstart

Add the script, use the tag.

<!-- import once -->
<script type="module" src="https://esm.sh/ferrite-player"></script>

<ferrite-player src="https://cdn.example.com/video/master.m3u8"></ferrite-player>

The default skin — play, seek, volume, quality, captions, PiP, fullscreen, live badge, keyboard shortcuts — is included. See the full API →
